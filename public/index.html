Authentication Information

This website uses JWT (JSON Web Token) for authentication. Instead of using a database, user data is stored in a local user.json file on the server. The user credentials (username and password) are saved securely, and you can sign in whenever you need using the authentication token generated on login.

Important Precautions

As this website uses a local user.json file to store user data for authentication, please be aware of the following:

  • Simultaneous logins: Multiple users should not log in at the same time on different devices, as this could corrupt the user.json file.
  • File integrity: The user.json file is critical for authentication. If it's damaged or deleted, you may lose access to your account.
  • Data loss: Since we're not using a database, we recommend you avoid storing sensitive information, as it may not be backed up.
